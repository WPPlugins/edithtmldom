== Description ==

Get DOM url or file and remove or replace contents in few minutes. 
This plugin has been designed on the basis of API Simple HTML DOM Parser of http://sourceforge.net/projects/simplehtmldom/
Complete Support visit this page http://ulmdesign.mediamaster.eu/edithtmldom/ and http://ulmdesign.mediamaster.eu/edithtmldom/support.html


== Installation ==

1. Upload folder 'editdomhtml' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the link "EditHtmlDom" in the admin section of your WordPress site and set the options, you choose the url and if you want replace div, section, header, footer etc. with a new code html or to leave blank to remove component.
4. Copy and paste the shortcode [modifycontents] in a post, page or widget-text where you want to display the url/contents of the page modified.
